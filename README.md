# 🚀 E-Commerce "Growth & Profit" Engine
An end-to-end data analytics project focused on Profitability, Seasonality, and Customer Behavior using SQL and Python.

## 📌 Project Overview
The goal of this project is to move beyond simple data counting and dive into **Relational Data Modeling**. By joining sales data with return records, I analyzed which product categories drive the highest net profit and identified seasonal trends to optimize inventory.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Libraries:** Pandas (Data Manipulation), Matplotlib/Seaborn (Visualization), SQLite3 (SQL Engine)
- **Database:** Relational SQL (Left Joins, Aggregations)

## 📊 Business Questions Answered
1. **Profitability vs. Returns:** Which categories are losing the most money due to product returns?
2. **Seasonality:** What are the peak months for sales, and when should the marketing budget be increased?
3. **Relational Integrity:** Successfully merged disparate datasets (Orders & Returns) using SQL LEFT JOINs to ensure no sales data was lost.
